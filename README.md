# 회생사건 자동조회 프로그램

대법원 「나의사건검색」에서 여러 회생사건을 **한 번에 자동으로 조회**하고,
진행내용·주요 일정을 정리한 엑셀 파일로 만들어 주는 프로그램입니다.

캡차(보안문자)도 자동으로 풀고, 사건별 진행상황과 주요 날짜
(신청일·보전처분일·개시결정일·시부인표·조사보고서·회생계획안·인가/폐지)를
자동으로 뽑아 **총괄표 한 장**으로 정리해 줍니다.

---

## 준비물

준비할 것은 딱 하나, **크롬(Chrome) 브라우저**입니다.
- 없으면 [google.com/chrome](https://google.com/chrome) 에서 설치하세요.

보안문자 처리에 필요한 인증은 프로그램에 이미 포함되어 있어,
**별도의 키 입력 없이 바로 사용**할 수 있습니다.

## 사용법

### 1단계 — 프로그램 실행

`회생사건조회.exe` 를 더블클릭하면 아래 화면이 열립니다.

`[스크린샷 2] 프로그램 메인 화면`

### 2단계 — 엑셀 표 붙여넣기 ⭐

엑셀에서 아래 **4개 열**을 순서대로 복사한 뒤,
프로그램 왼쪽 큰 칸에 **붙여넣기(Ctrl+V)** 합니다.

| 일련번호 | 차주명 | 법원 | 사건번호 |
|---------|--------|------|---------|
| EY-001 | 홍길동 | 대구회생법원 | 2026회단102 |
| EY-002 | 김철수 | 대구회생법원 | 2026간회단1012 |

- 맨 위 제목 줄(헤더)은 있어도 되고 없어도 됩니다. 자동으로 걸러냅니다.
- 붙여넣으면 아래 미리보기 표에 인식된 사건이 나타나고, "○건 인식됨"이 표시됩니다.

### 3단계 — 당사자명 확인

기본값은 **"아이"** 입니다. 조회할 당사자명(예: 농협, 신한 등)에 맞게 바꿔 주세요.

### 4단계 — 조회 시작

**[▶ 조회 시작]** 버튼을 누르면:

1. 결과를 저장할 위치를 물어봅니다. (원하는 폴더 선택)
2. 크롬 창이 자동으로 열리고, 사건을 하나씩 조회합니다.
3. 오른쪽 화면에서 **실시간 진행률 · 성공/오류 건수 · 진행 로그**를 볼 수 있습니다.

## 결과 파일 보는 법

엑셀 파일을 열면 **맨 앞 시트가 「회생차주 총괄표」** 입니다.
사건별 주요 일정이 한눈에 정리되어 있습니다.

| 열 | 내용 |
|----|------|
| 신청일 | 사건 접수일 |
| 재산보전처분일 | 보전처분결정일 |
| 개시결정일 | 회생절차 개시결정일 |
| 시부인표 제출일 | 시부인표가 제출된 날 |
| 조사보고서 제출일 | 조사보고서 제출일 |
| 회생계획안 제출일 | 회생계획안 제출일 |
| 인가일 / 폐지(종결)일 | 최종 결과 (인가·기각·취하 등) |

- **"미확인"(빨간 글씨)** 은 해당 단계가 아직 진행되지 않았거나,
  자동 인식이 어려운 경우입니다. → 그 사건의 **개별 시트**를 열어 직접 확인하세요.
- 총괄표 뒤에는 **사건별 상세 시트**가 하나씩 있습니다.
  진행내용 전체와 원본 글자색이 그대로 보존되어 있습니다.

⚠️ 개별 시트 확인 — 열람대상 완전성 검증

총괄표 뒤의 사건별 상세 시트에는 진행내용마다 열람대상(F열) 표시가 붙습니다.
Y 는 프로그램이 자동으로 골라낸 열람 요청 대상입니다
(신청서접수·시부인표·조사보고서·심문기일 등 정형화된 13개 항목).

중요: 자동 분류는 정형화된 항목만 Y로 잡습니다.
그 외의 결정·명령·기타 서류(예: 보전처분결정, 포괄적 금지명령, 각종 기타보고서 등)는
Y가 붙지 않고 빈칸으로 남을 수 있습니다.

따라서 열람 요청 전에는 Y만 보고 끝내지 말고, 빈칸(열람대상 없음) 행도 반드시 훑어보아
실제로 열람이 필요한 항목이 빠지지 않았는지 직접 확인해 주세요.

권장 검토 방법


개별 시트에서 8행의 필터(▽) 를 이용해 정렬·필터링합니다.
먼저 열람대상 = Y 행으로 1차 열람 대상을 확인합니다.
그다음 빈칸 행 전체를 눈으로 훑어, 결정문·명령·기타 제출 서류 중
열람이 필요한 것이 섞여 있지 않은지 점검합니다.
자동 표시(Y) + 직접 찾은 항목을 합쳐서 최종 열람 요청 목록을 작성합니다.



프로그램의 자동 분류는 1차 스크리닝 보조 도구입니다.
최종 열람 요청의 완전성은 반드시 사람이 확인해야 합니다.

또한, 해당 은행의 회생채권신고서는 열람대상에 포함되어 있지 않으므로, 반드시 별도로 채크해야 합니다.

## 자주 묻는 질문

**Q. 조회 도중 크롬 창을 닫아도 되나요?**
A. 안 됩니다. 조회가 끝날 때까지 그대로 두세요. 실수로 닫으면 [조회 시작]을 다시 누르면 됩니다.

**Q. "미확인"이 너무 많아요.**
A. 진행 중인 사건은 아직 그 단계에 도달하지 않아 정상입니다.
   완료된 사건인데 미확인이면, 그 사건 시트를 열어 직접 확인해 주세요.

**Q. 사건 몇 개는 "오류"로 나왔어요.**
A. 보안문자 인식 실패나 일시적 접속 문제일 수 있습니다. 해당 사건만 다시 조회해 보세요.

**Q. 몇 건까지 한 번에 되나요?**
A. 제한은 없지만, 건수가 많으면 시간이 오래 걸립니다. (사건당 약 10~20초)
